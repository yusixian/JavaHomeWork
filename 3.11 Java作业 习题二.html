1.问答题

（1）标识符是用来标识类名、变量名、方法名、类型名、数组名及文件名的有效字符序列称为标识符。标识符是由字母、数字、下划线和美元符号（$）组成。第一个字符不能为数字，标识符也不能是关键字或者true或false或null。false不可作为标识符
（2）关键字是具有特定用途或被赋予特定意义的一些单词，是系统的保留字。true和false不是关键字。boolean、int、double、float、short、byte
（3）逻辑类型：boolean
整数类型：byte、short（短整型）、int（整型）、long（长整型）
字符类型：char
浮点类型：float（单精度实数）、double（双精度实数）
（4）float常量在表示时后面必须有后缀f或F，而double常量在表示时后面可以有后缀d或D，但允许省略该后缀
（5）用length函数，如图所示
代码：

public class Main {
	public static void main(String[] args) {
		int[] a = new int[6];
		int[][] b = new int[5][6];
		System.out.println("一维数组a的长度为"+a.length);
		System.out.println("二维数组b中一维数组的个数为"+b.length);
	}
}

（4）下列程序标注的【代码1】和【代码2】的输出结果是什么

package test;
public class Main {
	public static void main(String[] args) {
		long [] a = {1, 2, 3, 4};
		long [] b = {100, 200, 300, 400, 500};
		b = a;
		System.out.println("数组b的长度：" + b.length);	//【代码1】
		System.out.println("b[0] = " + b[0]);			//【代码2】
	}
}

由于将a的引用赋给了b，所以b的长度实际是a的长度，而b[0]也即为a[0]
代码1输出 数组b的长度：4
代码2输出 b[0] = 1

（5）下列程序标注的【代码1】和【代码2】的输出结果是什么

public class Main {
	public static void main(String[] args) {
		int [] a = {10, 20, 30, 40}, b[] = {{1, 2}, {4, 5, 6, 7}};
		b[0] = a;
		b[0][1] = b[1][3];
		System.out.println(b[0][3]);	//【代码1】
		System.out.println(a[1]);			//【代码2】
	}
}

可见b[0]先是{1,2}，又被赋值为a的引用。则后一步操作即为将a[1]变为了7，故代码1输出为40，代码2输出为7

（2）编写一个Java应用程序，输出全部的希腊字母

查阅资料的得知
希腊字母（大小写各24个）：
大写：Α ~ Ω（对应Unicode码为：913 ~ 937，无930）
小写：α ~ ω（对应Unicode码为：945 ~ 969，无962）

public class Main {
	public static void main(String[] args) {
		for(int i = 913; i <= 937; ++i) {
			System.out.print((char) i);
		}
		System.out.println();
		for(int i = 945; i <= 969; ++i) {
			System.out.print((char) i);
		}
	}
}
